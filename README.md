# Consulting Landing Page for Validation

A professional landing page to validate your consulting business idea and collect early interest.

## What's Included

- Hero section with clear value proposition
- Problem/solution framework
- Service breakdown (Technical + Creative)
- Social proof elements
- Email collection form
- Mobile responsive design
- Professional gradients and animations

## Getting Started

```bash
npm run dev
```

Then open http://localhost:5173 in your browser.

## Customization Checklist

Before launching, customize these sections in `src/App.jsx`:

### Key Sections to Update
1. **Line 26-27:** Your name and branding
2. **Line 73-87:** Your experience stats
3. **Line 318:** Your email address
4. **Line 325:** Your LinkedIn URL
5. **Line 334:** Your Calendly link
6. **Line 349:** Footer info

## Deployment

```bash
npm run build
# Then deploy `dist` folder to Vercel, Netlify, or GitHub Pages
```

See full customization guide inside this README.
